# Week 10. 출력 친화적 설계 방법

## 🎯 이번 주 목표

이번 주의 목표는  
**설계를 ‘보기 좋은 모델’이 아니라 ‘제작 가능한 형상’으로 만드는 사고방식**을 익히는 것이다.

3D 프린팅에서는:
- CAD에서 가능한 형상 ≠ 출력 가능한 형상
인 경우가 많다.

이번 주에는
출력 실패를 줄이고,
후처리를 최소화하며,
구조적으로 안정적인 형상을 설계하는 원칙을 익힌다.

---

## 🧠 핵심 개념

### 1. 출력 방향은 설계의 일부다
출력 방향(Orientation)은
출력 품질과 강도에 큰 영향을 준다.

- 레이어 방향과 하중 방향의 관계
- 서포트 발생 여부
- 표면 품질 차이

형상 설계 시
출력 방향을 먼저 고려해야 한다.

---

### 2. 오버행과 브리징의 한계
FDM 출력에서는
공중에 떠 있는 형상이 큰 제약이 된다.

- 오버행 각도 제한
- 브리징 거리 한계
- 서포트 사용의 비용

가능하면
오버행을 피하고,
자연스럽게 쌓이는 구조를 만든다.

---

### 3. 두께와 강성의 균형
출력물의 강도는
단순히 두께를 늘린다고 해결되지 않는다.

- 리브(Rib) 추가
- 필렛/라운드 처리
- 내부 구조 설계

를 통해
강성과 출력 시간을 균형 있게 조절한다.

---

## 🛠️ 실습 / 작업 내용

### 1. 출력 실패 형상 재설계
- 기존 출력 실패 모델 선택
- 실패 원인 분석
- 형상 수정 방향 설정

---

### 2. 출력 방향 비교 실험
- 동일 모델을 다른 방향으로 슬라이싱
- 서포트량, 출력 시간, 표면 품질 비교
- 최적 방향 선정

---

### 3. 구조 개선 적용
- 리브 추가
- 불필요한 두께 제거
- 필렛 적용

출력 전/후 차이를 확인한다.

---

## 📁 산출물

- 수정된 CAD 모델 파일
- 출력 방향별 슬라이싱 결과
- 출력물 사진 비교

---

## ❗ 문제와 해결

### 문제 1. 서포트가 너무 많다
- 원인: 형상 설계 시 출력 방향 미고려
- 해결:
  형상을 분할하거나
  출력 방향을 재설계한다

---

### 문제 2. 출력물 강도가 약하다
- 원인: 레이어 방향과 하중 방향 불일치
- 해결:
  하중 방향에 맞게 출력 방향을 조정한다
